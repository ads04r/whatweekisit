"What Week Is It?" needs some term date data in order to function. This is all available from the Open Data site, data.soton.ac.uk. But to make this process as easy as possible, there is a command line PHP script in the /bin directory that downloads the correct data into the correct place automatically.

To download the data, simply run the file ./bin/download-data. Once you've done this, this site should then Just Work. If it doesn't, ensure the files current.ttl and all.ttl are present in the /var directory.

OPTIONAL: The download-data script will download all the data and parse out the bits it doesn't need, just for reasons of efficiency. This means that at the end of the current semester or term, the data will be out of date, and therefore this process must be repeated regularly. To ensure this happens, open your crontab file and add a line to make the script run once a day, preferably at a quiet time such as the early hours of the morning.
